## Getting Started / Contributing

1. Install npm dependencies

```bash
$ npm install
```

1. Create a .env file in the root of the project with the following values

```bash
NODE_ENV=
LOCAL_DATABASE_PASSWORD=
DB_USERNAME=
DB_HOST=
DB_TYPE=
DB_NAME=
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
